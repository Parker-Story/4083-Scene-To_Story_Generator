# Once Upon An Image

## Description
The Once Upon An Image is a collaborative AI project that transforms images into short, descriptive stories. 
It uses two AI agents: an **Image Agent** for detecting key objects and scenes, and a **Text Agent** for generating narratives based on the detected elements.

## Features
- Detects objects and scenes in images.
- Generates descriptive stories or captions.
- Modular design with separate agents for vision and language processing.
- Easy to extend with additional AI capabilities or datasets.

## Getting Started
1. Ensure Python 3.x is installed.
2. Install dependencies from `requirements.txt`.
3. Add sample images to the `data/sample_images` folder.
4. Run `main.py` to generate stories for the images.
